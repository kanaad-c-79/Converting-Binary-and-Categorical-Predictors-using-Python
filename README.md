# Converting-Binary-and-Categorical-Predictors-using-Python
# Description
We have a dataset with the following variables: Year, Age, Marital Status, Race, Education level, Region, Job Sector, Health, Health Insurance, Wage - with Wage being our target variable.

Converted the categorical variables from this dataset into numerical values.

Now that all the variables in our dataframe are numerical, I applied Linear Regression models to interpret how all the input variables correlated with Wage (target variable).
